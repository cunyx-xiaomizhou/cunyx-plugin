#### 安装教程

1.  进入云崽根目录
```bash
cd Yunzai-Bot
```

2.  拉取项目
```bash
git clone  --depth=1 https://gitee.com/cunyx/cunyx-plugin.git ./plugins/cunyx-plugin/
```

3.  重启机器人，生效控制面板
```bash
pnpm restart
```
