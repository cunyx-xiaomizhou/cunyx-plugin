#### 安装教程

1.  进入喵崽根目录
```bash
cd Miao-Yunzai
```

2.  拉取项目
```bash
# 使用Gitee拉取
git clone  --depth=1 https://gitee.com/cunyx/cunyx-plugin.git ./plugins/cunyx-plugin/
```
```bash
# 使用Github拉取
git clone  --depth=1 https://github.com/cunyx-xiaomizhou/cunyx-plugin.git ./plugins/cunyx-plugin/
```

3.  重启机器人，生效控制面板
```bash
pnpm restart
```
